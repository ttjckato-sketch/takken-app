# Takken OS v2.0 MVP 仕様書

## 開発方針
- **段階的アプローチ**: MVP → 段階的に機能追加
- **レスポンシブデザイン**: PC/スマホ両対応
- **フル統合**: 記憶支援 + 理解支援

---

## MVP実装機能（フェーズ1）

### 1. データ統合
- ULTIMATE_STUDY_DECK.json（メインデータ）
- OPTIMIZED_SRS_SYSTEM.json（SRS）
- ACTIVE_RECALL_SYSTEM.json（想起練習）
- ANALOGY_SYSTEM.json（アナロジー）
- STEP_DECOMPOSITION.json（ステップ分解）

### 2. 基礎UI/UX
- ホーム画面（ダッシュボード）
- ナビゲーション（ボトムタブ + サイドバー）
- レスポンシブレイアウト

### 3. コア学習機能
- **学習モード選択**:
  - 理解優先モード
  - 記憶優先モード
  - フル統合モード
- **カード表示**:
  - 基本情報
  - アナロジー表示
  - ステップ分解表示
  - 想起練習（ブランク埋め）
- **回答・評価**:
  - 自己評価（5段階）
  - 正解/不正解フィードバック

### 4. 進捗管理
- 学習統計
- SRSスケジュール
- 連続学習日数（Streak）
