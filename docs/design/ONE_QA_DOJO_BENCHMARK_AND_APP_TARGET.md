# ONE 1Q1A DOJO BENCHMARK AND APP TARGET

## 1. 競合ベンチマーク: 一問一答道場 (takken-siken.com)

| 機能 | 一問一答道場 | 当アプリの対抗策 |
|---|---|---|
| **問題数** | 5,231肢 (宅建のみ) | **約 5,875件** (宅建 3,875 + 賃貸 2,000) |
| **解説品質** | 網羅的だが、文章が長くポイントが掴みにくい | **構造化解説** (結論・理由・あてはめ) で秒速理解 |
| **学習単位** | 試験の「肢」単位 | **statement_items** (カード・肢) 単位 |
| **出題指定** | 分野・回数・重要度 | **分野別集中学習** + **FSRSによる忘却曲線管理** |
| **UI/UX** | クラシックなWebフォーム形式 | **モダンな学習カード形式** (スマホ最適化) |
| **強み** | 圧倒的な実績、履歴管理、無料公開 | **理解の深さ**、**短時間でのインプット効率** |

## 2. 当アプリの勝ち筋 (Value Proposition)

### A. 解説の「秒速理解」
一問一答道場は「読む」解説ですが、当アプリは「一目で入る」解説を目指します。
1. **結論第一**: 「なぜ〇か」ではなく「結論として〇である」を最初に出す。
2. **理由の分解**: 2〜3点の箇条書きで、論理のステップを明確にする。
3. **具体的あてはめ**: 「AがBに...」という問題文の具体名を解説に含める。

### B. 誤答後の「知識補修」
単に正解を確認するだけでなく、その論点の「構造化知識 (High Quality Input)」へワンタップでアクセスできる導線。

### C. 似た論点との「比較学習」
「35条書面と37条書面の比較」など、学習者が混同しやすいポイントを自動で提示。

## 3. statement_items (正本単位) の妥当性
- **総数: 約 5,875 件**
  - 宅建: 3,875 (カード化済み)
  - 賃貸: 2,000 (肢分解)
- **Dojo との差分理由**: Dojo は全過去問の全肢を網羅しているが、当アプリは「学習効率」を優先し、重複論点や重要度の低い古い問題を AI が選別・集約 (Card化) しているため。

## 4. UI要件 (Dojoを超える体験)
1. **問題文の最大化**: 学習中は問題文だけに集中。
2. **○×ボタンの巨大化**: 直感的なタップ操作。
3. **解説パネルのレイヤー化**: 結論は常に表示、詳細条文は「深掘り」時のみ。
4. **フィードバックの視覚化**: 正解時は緑、誤答時は赤のテーマカラーで脳に焼き付ける。

## 5. 今後やらないこと
- **多機能なランキング機能**: 競争より自己成長を優先。
- **掲示板・コメント機能**: 雑音を排除し、正確な DB 解説に絞る。
- **PC専用の複雑な管理画面**: スマホで完結する学習体験を優先。
