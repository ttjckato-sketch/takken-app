# Takken App v2.0-rc1 リリースノート (草案)

## 1. 概要
本バージョン (v2.0-rc1) は、宅建・賃貸管理士学習アプリの完成定義v2に基づく最初のリリース候補版です。「演習・補修・復習・分析・比較」の主要学習サイクルがMVPレベルで完結し、Fresh DB 環境での完全な再現性を実証しています。

## 2. 主な機能・改善
- **ActiveRecall (演習)**: 1024件の安全母集団（宅建）による〇×演習機能を安定化。
- **RepairPreview / Input Unit (補修)**: 誤答時に構造化された重要項目（23件）を即座に確認できる補修ループを構築。カテゴリの完全正規化を実施。
- **MemoryRecall (復習)**: FSRSアルゴリズムを用いた科学的な復習スケジューリングを統合。復元された高品質教材（23 KU / 46 MC）も対象。
- **比較学習 (比較)**: 混同しやすい類似論点を対比形式で整理する「比較学習ビュー」を新規実装。
- **初期学習者ガイダンス**: 履歴0件のユーザーに対し、ダッシュボード上で学習の進め方を動的にナビゲートする UI を追加。
- **データ基盤**: 破損教材の自律復元パイプライン（サイドカー方式）の確立。

## 3. DB固定値 (初期教材セット)
- 宅建ActiveRecall安全母集団: 1024件
- 構造化Input Unit: 23件
- 復元済み教材: 23 KU / 46 MC
- 学習履歴: 0件 (Clean Start)

## 4. 監査結果サマリー
- **Fresh DB再現性**: 100% (Playwrightによる自動検証済)
