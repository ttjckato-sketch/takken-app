## 3. 法的問題点

### 3.1 農地法3条の正確な解釈

**正しい解釈**:
- 農地法3条: 農地の権利移転（所有権移転、賃借権設定等）
- 適用場面: 農地を農地のまま使用する場合
- 手続: **原則として農業委員会の許可**
- 市街化区域内の扱い: **市街化区域内でも「許可」が必要**

**重要**: 農地法3条の「届出」は存在しない。3条は一律「許可」である。

### 3.2 農地法4条・5条の正確な解釈

**農地法4条**: 自己所有の農地を転用する場合
- 市街化区域内: **届出**で足りる（4条1項但し書き）
- 市街化区域外: 許可が必要

**農地法5条**: 他人の農地を取得・賃借して転用する場合
- 市街化区域内: **届出**で足りる（5条1項但し書き）
- 市街化区域外: 許可が必要

**重要**: 「市街化区域内で届出」は4条・5条の**転用場面**の特例である。

### 3.3 Pilot 001の誤り

**Pilot 001の解説**:
```
市街化区域内の農地（800㎡）について、所有権を移転する場合、
農地法第3条第1項の届出が必要である。
```

**問題点**:
1. **所有権を移転する場合** → 3条が適用される
2. **市街化区域内** → 3条の場合、市街化区域内でも「許可」が必要
3. **「届出が必要」** → 3条に「届出」は存在しない

**正しい解釈**:
```
市街化区域内の農地（800㎡）について、所有権を移転する場合、
農地法第3条第1項の「許可」が必要である。
```

**注**: is_statement_true_snapshot = true は正しいが、Pilot 001の解説文が誤っている。

## 6. 混同パターン分析

### 6.1 混同の構造

**Pilot 001の混同**:
```
所有権移転（3条の適用場面）
  ↓
市街化区域内（4条・5条の転用特例の適用場面）
  ↓
「届出で足りる」（4条・5条の転用特例）
```

**正しい構造**:
```
所有権移転 → 3条適用 → 市街化区域内でも「許可」が必要
自己所有転用 → 4条適用 → 市街化区域内は「届出」
他人所有取得転用 → 5条適用 → 市街化区域内は「届出」
```

### 6.2 混同の原因

1. **市街化区域内の特例を3条に適用した誤り**
   - 市街化区域内の「届出」特例は4条・5条の転用場面に限定される
   - 3条の権利移転には「届出」特例は存在しない

2. **「許可」と「届出」の区別の欠如**
   - 3条は「許可」
   - 4条・5条は市街化区域内で「届出」

3. **source_refsと解説の不整合**
   - source_refsは正しい条文（3条1項）を指している
   - しかし、解説内容は「届出」で誤っている
   - source_refs_alignment_okがtrueになっていたが、実際は不整合

## 8. 必要な修正

### 8.1 農地法3条/4条/5条の判定ルール明文化

**農地法3条**:
- 農地を農地のまま権利移転・賃貸借・使用貸借する場面
- 原則、農業委員会の**許可**
- 市街化区域内でも3条は**届出ではなく許可**
- 「3条 + 届出」は原則として疑義候補

**農地法4条**:
- 自己所有農地を自分で転用する場面
- 市街化区域外は原則許可
- 市街化区域内は**届出**で足りる（4条1項但し書き）

**農地法5条**:
- 他人の農地を取得・賃借して転用する場面
- 権利移転 + 転用
- 市街化区域外は原則許可
- 市街化区域内は**届出**で足りる（5条1項但し書き）

### 8.2 label_conflict_suspected検出強化

以下は疑義候補にする:

- 「3条」と「届出」が同じstatementに出る
- 「所有権移転」かつ「3条」かつ「届出」
- 「農地のまま」なのに4条/5条扱い
- 「転用目的」なのに3条扱い
- 「市街化区域内だから3条届出」とする説明
- 4条/5条の市街化区域届出特例を3条に適用している説明

### 8.3 source_refs alignment強化

source_refsが存在しても、根拠条文の解釈が正しいか確認する:

- 3条許可の根拠と、4条/5条届出の根拠を混同していないか
- supports_fieldで指定されたフィールドに根拠が正しく反映されているか
