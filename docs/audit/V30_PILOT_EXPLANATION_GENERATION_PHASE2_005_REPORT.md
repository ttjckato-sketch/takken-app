## 4. カテゴリ別詳細

### 4.1 Pilot 002: 農地法 3条

**カードID**: TAKKEN-SQ-AGRI-002
**問題文**: 市街化区域外の農地について、農地を農地のまま所有権移転する場合、農地法3条の許可が必要である。

**正解選択肢**: A（許可が必要）
**誤り選択肢**: B（届出が必要）, C（4条の許可）

**要点**:
- 市街化区域外の権利移転場面では3条の「許可」が必要
- 4条・5条の市街化区域内届出特例は適用外
- 選択肢Bに「3条 届出」のtrap検出あり

### 4.2 Pilot 003: 35条/37条書面

**カードID**: TAKKEN-SQ-35-003
**問題文**: 宅地建物取引業者は、契約成立後遅滞なく、宅建業法37条の規定に基づき契約内容を記載した書面を買主に交付しなければならない。

**正解選択肢**: A（37条書面交付）
**誤り選択肢**: B（35条書面交付）, C（契約成立前）

**要点**:
- 35条 = 契約成立前の重要事項説明
- 37条 = 契約成立後の契約内容記載書面交付
- 本件は「契約成立後」のため37条が適用

### 4.3 Pilot 004: 媒介契約

**カードID**: TAKKEN-SQ-BKAI-004
**問題文**: 専任媒介契約において、宅建業者は業務処理状況を宅建業法に規定する方法により依頼者に報告しなければならない。

**正解選択肢**: A（2週間に1回以上）
**誤り選択肢**: B（1ヶ月に1回以上）, C（3ヶ月に1回以上）

**要点**:
- 専任媒介契約 = 2週間に1回以上報告
- 専属専任媒介契約 = 4週間に1回以上報告
- 一般媒介契約 = 報告義務なし

### 4.4 Pilot 005: クーリング・オフ

**カードID**: TAKKEN-SQ-COOL-005
**問題文**: 宅地建業者の事務所等以外の場所で売買の申込みを受けた宅建業者は、買主に対し、書面によりクーリング・オフの告知をしなければならない。

**正解選択肢**: A（書面により告知）
**誤り選択肢**: B（口頭により告知）, C（買主が申出）

**要点**:
- 「告知」 = 宅建業者の義務
- 「申出」 = 買主の権利
- 事務所等外では「書面」による告知が必要

### 4.5 Pilot 006: 賃貸住宅管理業法

**カードID**: TAKKEN-SQ-CHIN-006
**問題文**: 賃貸住宅管理業者は、管理受託契約の締結時に、賃貸住宅管理業法67条に規定する重要事項説明書面を依頼者に交付しなければならない。

**正解選択肢**: A（書面交付）
**誤り選択肢**: B（業務管理者本人説明）, C（業務管理者以外可）

**要点**:
- 67条1項 = 業者による書面交付義務
- 67条2項 = 業務管理者本人記名押印
- 本人説明義務は67条2項のみ
